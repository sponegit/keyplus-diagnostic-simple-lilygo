### ⚡ 电气参数

| 特性                           | 详情     |
| ------------------------------ | -------- |
| USB 输入电压                   | 4.5~5.5V |
| 太阳能输入电压                 | 4.5 ~ 6V |
| 电池接口允许输入电压           | 4.2V     |
| MicroUSB/太阳能充电最大电流    | 500mA    |
| USB OUT                        | 1A       |

### ⚡ 电池保护器说明

| 特性               | 详情       |
| ------------------ | ---------- |
| 过压阈值           | 4.30V      |
| 欠压阈值           | 2.4V       |
| 过放电电流         | 3A         |
| 芯片名称           | [DW06D][2] |

### 按钮说明

* 首次上电或电池保护触发时，按下按钮将恢复电池输出功能。

### LED 说明

| 功能             | 颜色 | 状态                                                     |
| ---------------- | ---- | -------------------------------------------------------- |
| 太阳能指示灯     | 黄色 | 输入反接时灯常亮，输入正常连接时熄灭。                   |
| 电池指示灯       | 黄色 | 输入反接时灯常亮，输入正常连接时熄灭。                   |
| 5V 输出指示灯    | 绿色 | 5V 输出灯常亮，关闭时熄灭。                              |
| 充电指示灯       | 绿色 | 充满时灯常亮，充电时熄灭                                 |
| 充电指示灯       | 红色 | 充电时灯亮，充满时熄灭。                                 |
