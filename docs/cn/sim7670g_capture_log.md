# SIM7670G 日志抓取步骤


1. 升级到对应模块的最新版本。请观看 [升级视频](https://github.com/Xinyuan-LilyGO/LilyGO-T-A76XX/blob/main/docs/update_fw.md)


2. 升级完成后，启动日志抓取并捕获运行日志

3. [下载日志抓取记录器](https://drive.google.com/drive/folders/1-WwGcZ80KvAIHd8B9o6GRDqYoEec_qmY?usp=sharing)

4. 执行您的问题复现步骤，记录器将保存所有记录

5. 保存日志导出并发送给我们

以上步骤可以在 [此处](https://youtu.be/4vvNANhz86A) 的视频中查看
