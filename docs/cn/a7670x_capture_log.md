# A7670X 日志抓取步骤


1. 升级到对应模块的最新版本。例如，如果您使用 A7670G，请使用 A7670G 的升级包。如果您使用 A7670SA，请使用 A7670SA 的升级包。其他型号同理，因为日志抓取需要使用固件包中的 TxDB.txt 文件。请观看 [升级视频](https://github.com/Xinyuan-LilyGO/LilyGO-T-A76XX/blob/main/docs/update_fw.md)

2. 升级完成后，启动日志抓取并捕获运行日志

3. [下载日志抓取记录器](https://drive.google.com/drive/folders/115qZ_0FeM-abigEliNge3JbYbbcAuR2x?usp=sharing)

4. 执行您的问题复现步骤，记录器将保存所有记录

5. 保存日志导出并发送给我们


以上步骤可以在 [此处](https://youtu.be/31MUyhX5UHs) 的视频中查看
